# IP Policy — Dagoretti High School Community Hub

## Ownership

This project is a community-owned open-source repository maintained by Dagoretti High School alumni volunteers. No single entity owns the intellectual property.

## Licence

MIT Licence. See [LICENSE](../LICENSE).

You may fork, adapt, and redistribute this project with attribution. If you adapt it for another Kenyan school, please open-source your version.

## Community Data

Alumni profiles, KCSE records, and other community-submitted data belong to the individuals who submitted them. The platform is a custodian, not an owner.

Personal data is held in accordance with the Kenya Data Protection Act 2019.

## Attribution

If you build on this project, include a visible attribution line:

> Based on [Dagoretti Community Hub](https://github.com/dagoretti-community/hub)
