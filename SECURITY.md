# Security Policy

## Reporting a Vulnerability

If you discover an error:

DO NOT open a public issue.

Email directly to:
contact@aikungfu.dev

## Scope

This project handles community-submitted personal data (names, roles, locations).
It does not store passwords, payment information, or sensitive personal documents.

For the current CSV-based deployment, there is no authentication layer.
The production roadmap (§6 of the engineering spec) includes Supabase row-level
security and admin approval workflows before community submissions are displayed.

## Response

Security reports are acknowledged within 48 hours and triaged within 7 days.
